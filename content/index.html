Play your own RPG games anywhere

EasyRPG is a community project to create a free, open source, role playing game creation tool, compatible with RPG Maker 2000/2003 games.

Learn more about EasyRPG

Play our sample game

Cross-platform

Play games with EasyRPG Player

EasyRPG Player supports a wide range of platforms including Windows, Linux, macOS, mobile devices, consoles and more.
You can even play games in your browser!

Game creation

Create games with EasyRPG Editor

EasyRPG Editor gives you the tools you need to make all kinds of role-playing games.
Use our Tools to work with original data files.

Get involved!

Contribute code or assets

As an artist you can contribute to the OpenRTP project which is a collection of free (Creative Commons) assets. There are over 200 assets, and the list is constantly expanding.
